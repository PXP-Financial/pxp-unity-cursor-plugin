# Support

## What this repository supports

This repository supports the `pxp-unity` Cursor plugin for PXP Unity integrators.

Use this repository for:

- plugin bugs
- missing or unclear plugin guidance
- feature requests for new skills or commands
- marketplace listing issues related to this plugin

## Before opening an issue

Please include:

- your stack and framework
- the PXP product area involved, such as Transactions, 3DS, Checkout, Token Vault, Reporting, or Risk screening
- what you asked Cursor to do
- what happened instead
- sanitized examples of payloads, logs, or screenshots when relevant

Do not include:

- live API credentials
- raw card data
- sensitive customer data
- internal-only merchant information unless it has been sanitized

## Support path

- For plugin issues, open an issue in this repository.
- For PXP account, merchant setup, or commercial support, use the normal PXP support and onboarding channels.

## Security

If the issue could expose customer data, credentials, or a security weakness, do not file a public issue with sensitive details. Use your normal PXP security or support escalation path instead.
